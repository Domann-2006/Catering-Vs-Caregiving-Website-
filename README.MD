# Portfolio – Abolaji Abimbola Afolake
## Setup Instructions

### Folder Structure
```
/portfolio
├── index.html         ← Main portfolio website
├── style.css          ← All styles
├── script.js          ← All JavaScript
├── cv.html            ← Printable/downloadable CV
└── /images            ← Place your images here
    ├── profile.jpg        ← Main hero profile photo (recommended: 480×580px)
    ├── about.jpg          ← About section photo (recommended: 500×600px)
    ├── food1.jpg          ← Gallery: Premium food presentation
    ├── food2.jpg          ← Gallery: Traditional Nigerian cuisine
    ├── event1.jpg         ← Gallery: Event catering setup
    ├── kitchen1.jpg       ← Gallery: Kitchen in action
    ├── caregiving1.jpg    ← Gallery + caregiving section: Caregiver with patient
    ├── caregiving2.jpg    ← Gallery: Compassionate care
    ├── cert-caregiver.jpg ← Certifications: Caregiver certificate image
    ├── cert-firstaid.jpg  ← Certifications: First Aid certificate image
    └── cert-hygiene.jpg   ← Certifications: Food Safety & Hygiene certificate

```

### Image Fallbacks
All images have placeholder fallbacks using placehold.co if the image files
are not present. The site is fully functional and beautiful even without images.
Simply replace the image files with real photos to upgrade the experience.

### How to Use
1. Copy all 4 files (index.html, style.css, script.js, cv.html) into your /portfolio folder
2. Create an /images subfolder inside /portfolio
3. Add your real photos using the filenames listed above
4. Open index.html in any browser

### CV / Download
- The "Download CV" button links to cv.html
- cv.html has a built-in "Print / Save as PDF" button
- To create a PDF: open cv.html → click "Print / Save as PDF" → Save as PDF in browser print dialog
